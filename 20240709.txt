1. linux

- 리눅스 사용자 추적 하는 법
    whoami명령어를 통해 어떤 사용자가 어느시간에 접속하였는지 알 수 있다.
    who명령어를 통해 현재 본인이 어떤 사용자 인지 확인할 수 있다.

- linux에서 설치파일을 다운로드 하는 법
    해당 설치 명령어인 apt_get 명령어가 최신 상태를 유지하여야 더 많고 정확한 파일을 다운로드할 수 있다
    sudo 명령어를 통하여 root관리자에 권한을 잠시 빌려오는 명령어이다.
    명령어는 sudo apt get install update&upgrade와 sudo apt-get install update&upgrade이 있다. (apt-get이 아직까지는 더 많고 정확한 설치파일을 안내한다.)

- ifconfig를 통해 해당 ip주소를 확인
    명령어 다운로드는 sudo apt install net-tools

- 자바소스코드는 .java이며 컴파일된 바이트 코드는 .class이다.
    java개발에는 컴파일을 요구(javac)하며 jdk가 필요하고 java 실행에는 jre가 필요하다.  
    리눅스 상 선행 명령어는 sudo apt install openjdk-17-jdk-headless 이다.

- 파일과 디렉토리를 관리하는 법
    touch(빈파일 생성)
    mkdir(디렉토리 생성)
    tree (현재 디렉토리로부터 하위디렉토리까지의 구조를 트리구조로 출력)

- history를 통해 과거 사용한 명령어를 볼 수 있다.

2. 변수/생성자/메소드

- 변수는 데이터로 데이터,스택,힙 영역의메모리로 올라가게된다. 
     static 과 같은 전역 변수 또는 정적 변수의 경우 데이터 에 올라간다.
     new를 통해 사용자에 의한 동적할당으로 생성된 객체의 경우 힙 영역에 올라간다.
     지역변수 및 정적변수의 경우 스택 영역에 올라간다.

- static 변수
    객체로부터의 호출이 필요없는 경우 static을 사용하여 나타낸다.
    static은 클래스가 로드될 때 메모리 메소드영역에 할당되므로 모든 인스턴스를 공유하는 특성을 지니고 있다.

- 생성자
    객체사용에 필수이며 생략시 자동 추가 된다.

- 메소드
    반환 타입을 필요로 한다.
    로컬 변수 선언, 멤버 변수값과 변수값 출력을 한다.

3. lombok
- Getter, Setter등 대표적으로 사용되는 형태의 데이터 핸들링 구조를 자동 생성해준다.
    - Getter 멤버 변수값 반환
    - Setter 멤버 변수값 수정
    - NoAragsConstructor, AllArgsConstructor 파라미터를 필요하지 않는 / 파라미터를 모두 필요하는 구조를 생성
