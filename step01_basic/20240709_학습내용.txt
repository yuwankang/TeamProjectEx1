2. 데이터를 가공하는 방법	
	- 저장, 활용, 삭제
	- 기능 : 함수, 메소드
	
	- 두개의 숫자 받아서 + 한후에 연산 결과 반환하는 기능 구현 : 메소드
	
		반환타입 이름1(데이터1, 데이터2){
			데이터1 + 데이터2
			호출한곳으로 값을 반환
		}
		
		int 이름2{
			데이터1 - 데이터2
			호출한곳으로 int 값을 반환
		}
	
		python
			def 함수(파라미터):
				데이터1 - 데이터2
				호출한곳으로 int 값을 반환
		js
			function 함수(파라미터){
				데이터1 - 데이터2
				호출한곳으로 int 값을 반환
			}
